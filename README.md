# TaskBank_Csharp_app
プロジェクト名【TaskBank】

※ファイル名「HomeGamen」がmainFormです。

「概要」

このプロジェクトは日々溜まるタスクとお金を貯めたい若人を支援する目的として開発しました。

このミッションを達成するためにゲーミフィケーション要素を取り入れています。

また、本アプリはフリー画像や画像生成系AIを使用せずに全て自作の画像を使い制作しました。



「システムの内容」

モードごとにあったタスクを設定し、達成すればお小遣いが増えます。お金を使用した場合、自己申告でお小遣いを減らし管理するシステムです。

※タスクの報酬金額はユーザー側で設定可能です。


「ターゲット層」

習慣化・タスクの管理×貯金を両立させたい15～25歳。


「作成背景」

このプロジェクトを作成した理由は以下の通りです。

・日々溜まるタスクを達成するためのモチベーションを維持するシステムが作りたい

・貯金を楽しく続けたい

・データベースを実践的に学習するため


「技術スタック」

・Frontend:C#

・Backend:C#

・Database:SQLServer


「機能一覧」

・デイリー機能：習慣化したい時に使用する項目。1ヵ月に1回リセット(1ヶ月経っていなくても1000円消費でリセット可)。
1日1回まで達成できる。達成しなければペナルティでお小遣いが減っていく。ログインしていないとさらにお小遣いが減る。

・ノーマル機能：一日に何度も達成したい場合に使用する項目。

・ボーナス機能：一日に1回までしか達成したくない場合に使用する項目。

・ハード機能：達成するのが難しく、報酬金額を1000円以上に設定したい場合に使用する項目。

・Wishlist機能：タスクとは別にやりたいことを報酬として書くことができる項目。達成感を作るために履歴として残すことができる。

・ミッション機能：ノーマルとボーナスの項目から毎日ランダムに内容が表示され、報酬金額が通常より高くなる。

・キャラクター会話機能：時間帯や単語をランダムに変えることで同じ内容になりずらい。若い人向けアプリのため、かわいい絵柄のキャラクターで制作。

・音楽機能：mp3の形式であれば、どんな曲でも簡易的なプレイリストとして使うことができる。音楽のオンオフも保存されるため、必要ない人は止めておくことができる。

・お小遣い管理機能：タスクを達成するとお小遣いが増え、Wishlistなどでお金を使用するとお小遣いが減るシステム。

「工夫した点」

・楽しく貯金やタスクを達成してもらうために、ミッション機能やキャラクター会話機能、音楽機能などを取り入れました。
また、Wishlist機能で自身のやりたいこと、ほしいものを言語化し目的意欲の向上を図っています。

・できるだけボタン回数を減らすために、ユーザーが自然な動作(選択を解除するなど)をするときにデータベースへ保存していました。
また、本アプリは毎日することを推奨しているため、スタート画面をなくし起動を早くしていたり、デイリーやミッションはホーム画面からワンタップで達成できるようにしています。
加えてキャラクターのセリフも一定時間ごとに自動で変わるようにしています。

・項目を分けることでタスクごとの種類をわかりやすくしています。若い人向けのため、全体的に色は明るく彩度が薄いものを使用するように心がけていました。


「課題・今後の改善」

・PC上でないと起動できないため毎日開くのが困難な点が課題だと感じました。そのため、現在Web開発として再度制作中です。

・初めて触る人がどんなアプリかイメージしずらいことが課題だと感じました。ハテナマークから説明書が表示されますが、よりシンプルに開発することで理解
しやすくしようと思います。具体的には入力する際に選択式のモードで区別します。その後並び替えでタスクの種類をわけ、わかりやすさを維持しようと考えています。
